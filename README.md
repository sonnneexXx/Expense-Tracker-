# 💰 Expense Tracker - Трекер расходов

**Иванова Виолетта** 

## 📝 Описание программы

Expense Tracker - это графическое приложение для отслеживания личных финансов с удобным интерфейсом. Позволяет добавлять расходы, фильтровать их по категориям и датам, а также анализировать траты за выбранный период.

### Основные возможности

- ✅ **Добавление расходов** - сумма, категория и дата
- ✅ **Просмотр всех расходов** в удобной таблице
- ✅ **Фильтрация** по категории и диапазону дат
- ✅ **Автоматический подсчёт** суммы за выбранный период
- ✅ **Экспорт/импорт** данных в JSON
- ✅ **Удаление** отдельных записей или всех данных
- ✅ **Валидация** вводимых данных
- ✅ **Автосохранение** в JSON файл

## 📋 Требования

- Python 3.7 или выше
- Установленные библиотеки:
  - `tkcalendar` - для удобного выбора даты
